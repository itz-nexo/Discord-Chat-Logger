# 💬 Discord Chat Logger
A lightweight Discord message logger built with Discord.py-self.

Chat Logger monitors a specific Discord channel and automatically forwards messages, images, files, and user information to a webhook in real time. Messages are sent using the original sender's username and avatar, creating a clean and accurate archive of channel activity.

---
### Dislclaimer

This project uses discord.py-self which is considered a "self-bot" by Discord. Self-bots are prohibited by Discord's Terms of Service and can result in account limitations, suspension, or termination. I reccomend using an alternative account for this!

### Features

- 📩 Real-time message logging
- 🖼️ Image and attachment support
- 👤 Preserves usernames and profile pictures
- ⚡ Fast webhook-based forwarding
- 🎯 Channel-specific monitoring
- 🐍 Built with Python and Discord.py
- 📁 Useful for moderation logs and backups

## 📋 Requirements

- Python 3.9+
- Discord Bot Token
- Discord Webhook URL
